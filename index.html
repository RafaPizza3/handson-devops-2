Bem-vindo ao Projeto de Pull Request

Sobre o Projeto

Este projeto tem como objetivo ensinar o fluxo de trabalho do GitHub, incluindo a criação de branches, commits, Pull Requests e revisões de código.

Explicação do projeto

Um projeto que utiliza pull requests (PRs) no Git tem como objetivo organizar, revisar e integrar alterações no código de forma colaborativa e controlada. Essa prática é fundamental em equipes de desenvolvimento para garantir qualidade, padronização e rastreabilidade das contribuições ao código.

Funcionalidades

  • Visualização de paginas HTML simples.
  • Interação com formularios.
  • Design simples e responsivo.
